SPF_WCT is a free plugin for Wordpress that can work with stripe payment services.



== Description ==

・You can easily add a striped payment service to the "credit card payment settings" of the Welcart e-Commerce plugin.

・In order to use this plugin, you need an environment where you can use the welcart plugin.

・Update work such as adding functions including repairs will be irregular, but if you have any questions, 
  please contact us from the following URL.

== Installation ==

・In your WordPress admin panel, go to Plugins > New Plugin, search for "SPF_WCT" for WordPress and click "Install now ".


= MANUAL INSTALLATION =

1. Alternatively, download the plugin and upload the entire "spfwct" folder to the /wp-content/plugins/ directory.

2. Activate the plugin.

3. Payment form installation method
    https://ec.sys-creator.org/forums/forum/installation


== RECOMMENDED ENVIRONMENT ==

Welcart Ver.1.9.6 ～ 2.1.5
WordPress Ver.5.6 or greater
PHP 7.4
MySQL 5.6 or greater
Original domain and SSL (Shared SSL is not recommended)

※Please check the "readme.txt" included with the Welcart e-Commerce plugin.
